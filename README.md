# Inverse Contrast - Dark Mode & Color Inverter Chrome Extension

Inverse Contrast is a lightweight, easy-to-use Google Chrome extension designed for night owls, developers, and readers. It instantly inverts the colors of any active browser tab, providing a high-contrast dark mode experience to reduce eye strain and improve web accessibility.

Toggle between Dark, Light, and Dark again with a single click or a custom keyboard shortcut!

## 🌟 Features
* **Instant Color Inversion:** Turn any blindingly white webpage into a soothing dark theme instantly.
* **Keyboard Shortcut Support:** Seamlessly toggle contrast without taking your hands off the keyboard.
* **Lightweight & Fast:** Runs purely locally with zero impact on your browser's speed or performance.
* **Privacy First:** No data tracking, no ads, just a simple color inverter.

## 🚀 How to Install (Developer Mode)

You can install this extension directly from the source code in just a few quick steps.

**Step 1: Download the Source Code**  
Download the repository files (excluding the README) and extract them somewhere on your computer (e.g., `C:\Users\<user>\Documents\Code\Inverse-Contrast`).  
*⚠️ Note: Do not delete this folder after installation, unless you want to delete the extension.*  
<img width="600" alt="Extract Extension Folder Structure" src="https://github.com/user-attachments/assets/f691207c-beb7-4709-9e5b-7d30d25bb144" />

**Step 2: Open Chrome Extensions**  
Navigate to your Manage Extensions page by clicking the puzzle icon in your browser or typing `chrome://extensions/` in your address bar.  
<img width="300" alt="Manage Chrome Extensions Menu" src="https://github.com/user-attachments/assets/4770c55f-992a-4260-8baa-2d2669dcae9c" />

**Step 3: Enable Developer Mode**  
Toggle the **Developer mode** switch located in the top right corner of the extensions page.  
<img width="233" alt="Turn on Developer Mode in Chrome" src="https://github.com/user-attachments/assets/3daa03b7-42de-432c-b916-66a6e60c65f8" />

**Step 4: Load Unpacked**  
Click the **Load unpacked** button that appears in the top left corner.  
<img width="211" alt="Load Unpacked Extension Button" src="https://github.com/user-attachments/assets/b75404a5-2da5-4fa3-8811-df18d99b6759" />

**Step 5: Select Your Folder**  
Select the folder where you extracted the extension files in Step 1.  
<img width="600" alt="Select Extension Folder Directory" src="https://github.com/user-attachments/assets/cafa6600-204a-4001-b3e0-7cfa6a4874af" />

**Step 6: Pin & Use!**  
The extension is now active! Pin it to your toolbar for easy access. Just click the icon whenever you want to invert a webpage's colors.

**Step 7: Set a Keyboard Shortcut (Optional)**  
For maximum productivity, assign a custom keyboard shortcut via the Chrome Extensions shortcut menu.  
<img width="229" alt="Chrome Extension Keyboard Shortcut Setup" src="https://github.com/user-attachments/assets/f2248387-832a-4c68-b4f9-fd990d32dad3" />
